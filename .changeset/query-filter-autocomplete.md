Registry-driven autocomplete for the `?q=` filter input (ADR 0047, slice 3): focusing the filter shows an operator cheat sheet, typing `key:` lists that key's registered values (`highlight:` with color swatches), and `#` suggests your tag corpus. ArrowUp/Down + Enter/Tab or a click inserts the term; Escape now closes the suggestion popover first, then the input, then the filter. A new plugin operator appears in the suggestions for free — nothing hand-maintained.
